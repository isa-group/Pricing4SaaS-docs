- **required**
- Field type: `seq` of plan names

Your addon is available for the plans that you specify in this list.

```yaml
plans:
  SILVER:
  GOLD:
  PLATINUM:
    # ...
addOns:
  RUBY:
    availableFor:
      - GOLD
      - SILVER
```
